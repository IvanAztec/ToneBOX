# SaaS Factory OS v3 - Antigravity

## Golden Path
- Framework: Next.js 15 (App Router)
- Auth/Database: Supabase
- Styling: Tailwind CSS
- AI: Vercel AI SDK + OpenRouter

## Estructura de Archivos (Feature-First)
- src/features/[feature-name]/
- src/components/shared/
- src/lib/supabase/

## Reglas de Oro
1. No inventar ruedas: Usar el Golden Path.
2. Archivos < 500 líneas.
3. Typescript estricto (No 'any').
4. Antes de codear una feature compleja, crear un PRP (Product Requirements Proposal).
